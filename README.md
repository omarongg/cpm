#### Tested on:

- Android (**Termux**).
- Windows
- Linux

## Installation
### Android (Termux):
- `pkg install git`
- `pkg install python python-pip`
- `git clone https://github.com/cpmnuker/cpmnuker.git`
- `cd cpmnuker`
- `pip install -r requirements.txt`
- `python main.py`

### IPhone (ISH):
- `apk add git`
- `apk add python3`
- `apk add py3-pip`
- `git clone https://github.com/cpmnuker/cpmnuker.git`
- `cd cpmnuker`
- `pip install -r requirements.txt`
- `python3 main.py`
